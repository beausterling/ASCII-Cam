- Dark background (`#000000` or `#0d1117`) for body

- No animations (deferred per user decision)

**js/config.js:**
- Export a `config` object with placeholder settings:
  - `charSet: '@%#*+=-:. '` (standard ASCII character ramp)
  - `resolution: 8` (grid cell size in pixels)
  - `theme: { background: '#000000', text: '#00ff00' }` (for future ASCII output)
- Add inline comments explaining what each config value controls

**js/renderer.js:**
- Import nothing (placeholder -- will use p5.js in Phase 2)
- Export `initRenderer()` function that logs "Renderer placeholder - will use p5.js in Phase 2"
- Add inline comments explaining this is a placeholder for the p5.js sketch

**js/analyzer.js:**
- Import `config` from `'./config.js'` (explicit .js extension required)
- Export `analyze(pixels)` function that logs "Analyzer placeholder" and returns `config.charSet[0]`
- Add inline comments explaining this will handle brightness/ASCII mapping

**js/main.js:**
- Import `config` from `'./config.js'`
- Import `initRenderer` from `'./renderer.js'`
- Import `analyze` from `'./analyzer.js'`
- Configure figlet font path: `figlet.defaults({ fontPath: 'https://cdn.jsdelivr.net/npm/figlet@1.10.0/fonts/' })`

  - Call `figlet('ASCII CAM', { font: 'Standard' }, callback)` to generate the logo

  - Include error handling: if figlet fails, fall back to plain text "ASCII CAM"
  - Log initialization message to console

**# ASCII Cam**
One-line description: A real-time webcam-to-ASCII art web app built with vanilla JavaScript and p5.js.

**## Features** (brief, current state)
- Figlet ASCII logo splash page
- ES module architecture (main, renderer, analyzer, config)
- Dark theme with Bootstrap 5
- Deployable as a static site (no build step required)

**## Quick Start**
1. Clone the repo
2. Start a local server (ES modules require HTTP, not file://):
   - Option A: `python3 -m http.server 8000`
   - Option B: `npx serve .`
3. Open `http://localhost:8000` in your browser

**## Development**
- `npm install` to set up ESLint and Prettier (optional dev tooling)
- `npm run lint` to check code style
- `npm run format` to auto-format all files
- `npm run dev` to start local development server

**## Deployment**
- Deployed via GitHub Pages from main branch
- HTTPS required for webcam access (provided by GitHub Pages)
- Push to main branch to deploy automatically

**## Project Structure**
Show the file tree with brief descriptions of each file's purpose.

- p5.js -- Creative coding library (webcam capture + canvas rendering)
- Bootstrap 5 -- Responsive layout and dark theme

- Vanilla JavaScript with ES modules

**## Known Limitations**
- ES modules require a local HTTP server (cannot open index.html directly via file://)
- Webcam features require HTTPS (GitHub Pages or localhost)
